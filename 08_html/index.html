Супер-пупер селектора

selectors

Селектора нужны для того, чтобы не устраивать хаоса в каждом отдельном .html файле, когда у вас намечается большой проект с сайтом со множеством страниц. Селектора пишутся в отдельном .css файле и его можно будет спокойно использовать для всех страниц типа .html.

В общем... Селектора бывают:

  • по названию элемента, то бишь, по тегам, таким как hx, p, a, etc.
  • по классу, что был создан пользователем (адмнистратором?), для применения к всевозможным элементам;
  • по id; тоже самое, что и класс, но в этом случае многоразовое использование (в рамках одного файла) не приветсвуется;
  • по нескольким классам, для упрощения стилизации сайта;
  • по вложенности; также для упрощения, но с другой стороны.

И, разумеется, так далее.

Благодаря секторам, можно с лёгкостью задать разным частям текста разные цвета.
